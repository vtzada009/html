tags semanticas em html 5

as tags semanticas são elementos HTML que descrevem claramente o significado do conteudo que elas envolvem.elas ajudam a melhorar a acessibilidade, a SEO e a estrutura do documento. aqui estao algumas das principais tags semanticas

  • <header>: define o cabeçalho de um documento ou seção.
  • <nav>: define um conjunto de links de navegação.
  • <mains>: define o conteudo principal do documento.
  • <section>: define uma seção generica de um documento.
  • <articlei>: define um conteudo primcipal do documento.
  • <aside>: define um conteudo relacionado,mas separado do conteudo principal,como barra lateral.
  • <footer>:define o rodape de um documento ou seção.

usar essas tags semanticas ajuda os motores de busca a entender melhor o conteudo do seu site a melhor a experiencia do usuario,esspecialmente para aqueles que ultilizam leitores de tela.
