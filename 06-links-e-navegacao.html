links de navegação html 5

link para site externo

link externo aponta para um endereço (URL)fora do seu site ou do servidor onde seus arquivos estão hospedados

como ele funciona

para que o navegador saiba que deve sair do seu dominio,atributo href do elemento <a >deve conter o endereço completo do site externo,incluindo o protocolo (http:// ou https://).

atributo target="_blank"

o atributo target="_blank"é usado para abrir o link em uma nova aba ou janela do navegador, permite que o usuario mantenha a pagina atual aberta enquanto visita o site externo.
